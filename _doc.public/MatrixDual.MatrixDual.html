Module MatrixDual.MatrixDual

module MatrixDual: sig .. end
type t = float Stdlib.Array.t Stdlib.Array.t

t is dual numbers represented as a matrix. For example, the dual number a + be is represented as [[a, b][0, a]].

val get_last_dual : t -> float

get_last_dual t returns the element at the upper right-most corner.

val make_matrix : int -> float -> t

make_matrix i input forms an upper-triangular Toeplitz matrix of dimensions i x i given the input number input.

val make_scalar : int -> float -> t

make_scalar dim s t makes a scalar s of dimension dim.

val matrix_add : t -> t -> t

matrix_add t1 t2 is the summation of two matrices.

val matrix_sub : t -> t -> t

matrix_sub t1 t2 is the subtraction of two matrices.

val matrix_div : t -> t -> t

matrix_div t1 t2 is the division of two matrices. Division is actually represented naively by a scalar fraction, since we will not be encountering anything more advanced in doing derivatives. Requires: t1 is a scalar matrix as generated by make_scalar.

val matrix_mult : t -> t -> t

matrix_mult t1 t2 is the multiplication of two matrices.

val matrix_power : t ->
       t -> int -> t

matrix_mult orig t n is the exponentiation of matrix t to the power n.
